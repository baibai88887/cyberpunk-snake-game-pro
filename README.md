# 赛博朋克贪吃蛇游戏

一个采用赛博朋克风格设计的经典贪吃蛇网页小游戏，具有现代化的视觉效果和沉浸式的游戏体验。

## 🎮 游戏特点

### 视觉设计
- **赛博朋克风格**：深色背景配合霓虹色彩（青色、粉色为主）
- **霓虹效果**：所有游戏元素都带有发光效果和阴影
- **故障艺术**：随机触发的屏幕故障效果
- **动态背景**：移动的网格线和颜色脉冲动画
- **扫描线效果**：复古CRT显示器的扫描线效果
- **静态噪点**：轻微的电子设备噪点效果

### 音效系统
- **赛博朋克风格音效**：使用合成器生成的电子音效
- **多层次音效**：游戏启动、吃食物、升级、游戏结束等不同音效
- **频率扫描**：动态变化的音效频率，增强科技感

### 游戏功能
- **经典玩法**：控制贪吃蛇移动，吃食物增长
- **等级系统**：随分数提高，游戏速度增加
- **计分系统**：实时显示当前分数和等级
- **游戏控制**：方向键控制，空格键暂停
- **响应式设计**：适配不同屏幕尺寸
- **游戏状态**：支持开始、暂停、继续、重新开始

## 🕹️ 游戏玩法

### 操作方式
- **方向键**：控制贪吃蛇的移动方向（上、下、左、右）
- **空格键**：暂停/继续游戏
- **按钮控制**：通过界面按钮控制游戏状态

### 游戏规则
1. 使用方向键控制贪吃蛇移动
2. 吃到粉色食物增加分数（+10分）和蛇的长度
3. 每50分升一级，游戏速度增加
4. 碰到边界或自身身体游戏结束
5. 可以随时暂停游戏

## 🛠️ 技术实现

### 技术栈
- **前端技术**：HTML5 + CSS3 + JavaScript (ES6+)
- **游戏渲染**：Canvas API
- **音效系统**：Web Audio API
- **样式框架**：原生CSS3（响应式设计）

### 架构设计
- **模块化结构**：清晰的代码组织，便于维护
- **游戏循环**：基于setInterval的游戏主循环
- **状态管理**：完整的游戏状态控制
- **事件驱动**：基于DOM事件的用户交互处理

## 📦 安装与运行

### 环境要求
- Node.js >= 14.0.0
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动项目

#### 开发模式
```bash
npm start
```
- 启动本地开发服务器
- 自动打开浏览器
- 访问地址：http://localhost:8000

#### 开发模式（无缓存）
```bash
npm run dev
```
- 启动本地开发服务器
- 禁用浏览器缓存
- 自动打开浏览器
- 适合开发调试

#### 构建项目
```bash
npm run build
```
- 生成生产版本
- 目前仅输出提示信息

### 传统方式（可选）

如果没有Node.js环境，也可以使用Python启动本地服务器：
```bash
python3 -m http.server 8000
```
然后在浏览器中访问：http://localhost:8000

## 🎨 设计风格

### 颜色方案
- **主色调**：霓虹蓝 (#00ffff)、霓虹粉 (#ff00ff)
- **背景色**：黑色到深紫色的渐变
- **文字色**：霓虹蓝为主，霓虹粉为辅
- **边框色**：发光的霓虹色边框

### 视觉元素
- **直角设计**：避免圆角，采用锐利的直角
- **等宽字体**：使用Courier New等宽字体，增强科技感
- **发光效果**：所有元素都带有阴影和发光效果
- **动态效果**：缓慢变化的颜色和位置
